Introduction

This is the documentation of an xml project, that presents a layout of a mobile's device settings page.
The tabs and contents that consist this settings page are based on a mobile device and are not a representation of the real layout.


Methodology

Project was made with NetBeans IDE 8.2.
The presentation of JSON output was made with Google Chrome browser
and lastly Github platform was used for the submission of the project.
Project implemented with XML, XMLSchema, XSLT and JSON.


Work content

Here we analyze the content of each file.

Content Description
XML.xml XML file is the first file that starts the making of the settings layout. Here are located all the components that give layout his shape.
XMLSchema.xsd XML Schema describes and validates the correctness of XML's components.
XSLT.xsl XSLT is used for the transformation of XML file to JSON.
JSON.json JSON format is chosen to be used as an output of the transformation of XML.

Elements description

  • Main element of our layout is "Settings", that contains all components.
  • Layout is splitted into three main subelements about "Device", "Personal" and "System" settings.
    "Device" tag has settings about:
    • "Sounds"
    • "Display"
    • "Memory"
      • Some of the elements inside "Sounds" element are volume,ringtone,message,alarm and vibrate.
      Some of the elements inside "Display" element are brightness,wallpaper,theme,font and screensaver.
      Some of the elements inside "Memory" element are sd and internal.
  • "Personal" tag has settings about:
    • "Apps"
    • "Users"
    • "Privacy"
      • "Apps" element contains three records splitted with ID. Subelements are title,size,location and installedon.
      "Users" element contains two records splitted with ID. Subelements are account,name and lastlog.
      Some of the elements inside "Privacy" element are simpassword and lockscreen that contain attributes activesim and activelock.
  • "System" tag has settings about:
    • "Date"
    • "Time"
    • "Language"
    • "About"
      • "Date" element contains attribute dt. Subelement is formd.
      "Time" element contains attribute tm. Subelement is formt.
      Some of the elements inside "Language" element are phone and keyboard.
      Some of the elements inside "About" element are brand (that contains attributes model,cpu,gpu and ram)s,osversion and status.
      Some of the elements inside "Status" element are battery,mnumber,screensize,network and imei.


Transformation description

  • XML transformation into XMLSchema.
    • Description of XML components to xsd and validation.
  • XML transformation into the XSLT file and production of JSON output.
